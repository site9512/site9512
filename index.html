SCP-173 - The Sculpture

Special Containment Procedures

Locked container, minimum 3 personnel present, constant eye contact required.

Description

Concrete statue that moves when unobserved. Extremely hostile.

SCP-096 - The Shy Guy

Special Containment Procedures

Sealed airtight container. Viewing face causes violent pursuit.

Description

Humanoid entity that becomes enraged when its face is seen.

SCP-049 - The Plague Doctor

Special Containment Procedures

Contained in humanoid cell, biohazard protection required.

Description

Medieval plague doctor entity that "cures" victims by killing them.

SCP-682 - Hard-to-Destroy Reptile

Special Containment Procedures

Reinforced chamber with electric fences and acid suppression.

Description

Highly adaptive, intelligent, hostile reptilian entity.

SCP-035 - The Possessive Mask

Special Containment Procedures

Hermetically sealed glass container, gloves and face shields required.

Description

Porcelain mask causing possession of wearer.

SCP-055 - Unknown

Special Containment Procedures

Keep secured; information about SCP-055 is unknowable.

Description

An object that resists all attempts to be remembered or described.

SCP-093 - Red Sea Object

Special Containment Procedures

Stored in secure locker. Used to access alternate dimensions via mirror.

Description

A red disc allowing access to alternate worlds when held against a mirror.

SCP-1471 - MalO ver1.0.0

Special Containment Procedures

Mobile app anomaly; containment not feasible, monitor subject behavior.

Description

A mobile app that manifests a humanoid figure in users' peripheral vision.

SCP-106 - The Old Man

Special Containment Procedures

Contained in a sealed chamber with corrosion-resistant materials.

Description

An old humanoid that can pass through solid matter and corrode materials.

SCP-008 - Zombie Plague

Special Containment Procedures

Stored in secure biohazard container. All samples incinerated after testing.

Description

A prion causing zombification in infected humans.

SCP-999 - The Tickle Monster

Special Containment Procedures

Contained in a secure playpen; allowed interaction with personnel.

Description

A friendly, gelatinous entity that induces happiness.

SCP-079 - Old AI

Special Containment Procedures

Contained on isolated computer hardware with network restrictions.

Description

An artificial intelligence with self-awareness and hostile tendencies.

SCP-1048 - Builder Bear

Special Containment Procedures

Contained in a reinforced humanoid containment cell.

Description

A teddy bear that creates copies from strange materials, often violent.

SCP-1782 - Taboo Tapes

Special Containment Procedures

Contained with other SCP media in a soundproof vault.

Description

A VHS tape that shows forbidden images not normally visible to humans.

SCP-131 - The Eye Pods

Special Containment Procedures

Allowed to roam containment corridors with supervision.

Description

Two small, friendly creatures that follow personnel around.

SCP-294 - The Coffee Machine

Special Containment Procedures

Contained in secure break room; monitored for anomalous output.

Description

A vending machine dispensing any liquid requested via text input.

SCP-963 - Immortality Mask

Special Containment Procedures

Stored securely, access limited to senior personnel.

Description

A mask granting the wearer's consciousness immortality by possession.

SCP-147 - The Old Lighthouse

Special Containment Procedures

Lighthouse is contained on-site with restricted access.

Description

A lighthouse that affects time and spatial perception around it.

SCP-343 - God

Special Containment Procedures

Currently considered safe and cooperative; no containment required.

Description

An entity claiming to be God; displays omniscience and benevolence.

SCP-914 - The Clockworks

Special Containment Procedures

Contained in a secured mechanical chamber with controlled access.

Description

A large clockwork machine capable of refining or degrading objects placed inside.

SCP-261 - The Vending Machine

Special Containment Procedures

Contained in a secure cafeteria; monitored for anomalous items dispensed.

Description

A vending machine that dispenses items unrelated to requested selections.

SCP-1312 - The Snowman

Special Containment Procedures

Contained in a temperature-controlled room to prevent melting.

Description

A snowman figure with minor reality-warping abilities.

SCP-1230 - Encyclopedia of Absolute Truth

Special Containment Procedures

Contained in a secure library, only authorized personnel may access.

Description

A book that imparts absolute knowledge and truth on any subject queried.

SCP-9512-2 - The Black Cloud Entity

Special Containment Procedures

Contained within a sealed environment using advanced mutagenic inhibitors.

Description

A black cloud entity capable of mutating organisms and SCPs, aggressively seeking to consume its own species.

About Our SCPs

Who Are Our SCPs?

Our SCPs represent a wide range of anomalous objects, creatures, and phenomena that defy the natural laws of our world.

Containment Philosophy

We prioritize secure containment, minimal personnel exposure, and detailed study to understand these anomalies.
